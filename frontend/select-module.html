BNM Workbench

A platform for hotspot modeling using Bipartite Network Modeling (BNM).

About This Workbench

This workbench is developed under the BNM Workgroup and is based on the Bipartite Network Modeling framework (BNM). It provides a unified interface for uploading, managing, and analyzing spatial disease hotspot data. This platform builds upon the following references:

  • BNM Framework: A methodology framework for bipartite network modeling
    Liew et al., 2023, Applied Network Science
    DOI: 10.1007/s41109-023-00520-7
  • Dengue Module: Modeling dengue hotspots with bipartite network approach
    Koh et al., 2018, ICCST 2017, Lecture Notes in Electrical Engineering
    DOI: 10.1007/978-981-13-0528-7_20
  • COVID-19 Module: Modelling COVID-19 hotspots using bipartite network approach
    Hong et al., 2021, Acta Informatica Pragensia
    DOI: 10.18267/j.aip.159
